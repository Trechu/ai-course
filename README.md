# Setup

#### Install poetry

```sh
curl -sSL https://install.python-poetry.org | python3 -
```

#### Create virtual env and pull dependencies

```sh
poetry install
```

#### Activate virtual env

```sh
poetry shell
```

#### Exit virtual env

```sh
exit
```

#### Delete virtual env

```sh
poetry env list
```

```sh
poetry env remove <environment-name>
```
